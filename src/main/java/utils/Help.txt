Приложение помогает организовать расчет характеристик распределения спиновой плотности в органических радикалах.
В текущей версии реализован расчёт кулоновского интеграла спиновой плотности [].

Для расчёта характеристик, спиновая плотность должна быть предоставлена в текстовом файле, для которого выполнен ряд требований:
1) В каждой строке приведены четыре числа: три координаты (x, y, z) и значение спиновой плотности;
2) Сетка равномерная, шаг у разных осей может быть разным;
3) Строки упорядочены так, чтобы сначала изменялись значения по оси z, затем по оси y и наконец по оси x.

Для ускорения вычислений, после перенормировки точки со значением, указанным в настройках, отсеиваются.
